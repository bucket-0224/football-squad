# ⚽ FC Management

**FC Management**는 나만의 축구단을 만들어 실시간 대전을 벌이는 웹 게임입니다.

실제 8개 리그(EPL·라리가·분데스리가·세리에 A·리그 1·MLS·브라질레이랑·K리그 1)의 클럽과 실선수 명단으로 스쿼드를 꾸리고, 카드 팩에서 선수를 뽑고, 강화하고, 이적 협상으로 영입해 다른 유저와 90분 경기를 치릅니다.

## 핵심 기능

- **실시간 탑뷰 대전** — 90분 경기가 약 1분의 실시간 탑뷰 중계로 재생됩니다. 오프사이드·코너킥·프리킥 수비벽·VAR·퇴장까지 실제 EPL 규칙 기반으로 연출됩니다.
- **스쿼드 빌딩** — 포메이션 5종, 전술 4종, 포지션 적합도·케미 시스템.
- **카드 팩 & 강화** — 브론즈~스페셜 팩에서 선수를 뽑고, 최대 +5까지 강화합니다. 80+ 카드는 워크아웃 세리머니가 나옵니다.
- **이적 협상** — 구단 이적료 합의 → 개인 보너스 합의의 2단계 협상.
- **작전 타임** — 경기 중 팀당 2회, 포메이션 변경·포지션 스왑·교체(경기당 5명).
- **관전 모드** — 진행 중인 다른 유저의 경기를 실시간으로 관전합니다.
- **승부 예측** — 실제 리그 경기 결과를 예측하고 코인을 받습니다.
