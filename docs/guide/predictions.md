# 승부 예측

**실제 리그 경기**의 결과를 예측하고 코인을 받습니다.

## 대상 경기

8개 리그(EPL·라리가·분데스리가·세리에 A·리그 1·MLS·브라질레이랑·K리그 1)의 **현재 라운드 전체**가 리그 구분 없이 킥오프 일정순으로 나열됩니다.

## 참여 방법

1. 킥오프 전 경기에서 **홈 승 / 무승부 / 원정 승**을 선택합니다.
2. 선택 사항으로 **정확한 스코어**까지 예측할 수 있습니다.
3. 킥오프 후에는 베팅이 마감되고, 실제 경기 종료 후 자동 정산됩니다.
