# 스쿼드 & 강화

## 스쿼드 편성

- **포메이션**: 4-3-3 / 4-4-2 / 4-2-3-1 / 3-5-2 / 4-1-4-1
- **전술**: 공격적 · 균형 · 수비적 · 역습 (역습은 공격적 상대에게 보너스)
- 슬롯을 눌러 선수를 배치하고, **자동배치**는 정포지션 우선(fit-first)으로 베스트 XI를 짭니다.

## 포지션 적합도

카드는 배치된 슬롯의 포지션으로 **변환**되어 표시되며, 원래 포지션과의 거리만큼 OVR이 감소합니다.

| 관계 | 감소 |
| --- | --- |
| 동일 포지션 | 0 |
| 같은 라인 (예: ST↔RW) | −2 |
| 인접 라인 (예: CAM↔ST) | −6 |
| 반대 라인 (예: CAM→RB) | −10 |

골키퍼는 GK 슬롯에만, GK 슬롯엔 골키퍼만 배치할 수 있습니다.

## 선수 강화

보유 선수 카드의 ⚡ 버튼으로 강화합니다.

- 단계당 **OVR과 모든 능력치 +1**, 최대 **+5**
- 성공 확률: 100% → 80% → 60% → 40% → 25%
- 비용은 선수 시장 가치에 비례하며, **실패해도 단계는 유지**됩니다(코인만 소모)
- 강화는 평점·시뮬레이션·탑뷰 움직임에 전부 반영되고, 카드에 보라색 `+N` 배지가 붙습니다
- 판매하거나 클럽 변경으로 떠나면 강화도 함께 사라집니다

## 클럽 변경

승점 250으로 베이스 클럽을 바꿀 수 있습니다. 기존 클럽 선수단은 떠나지만(뽑은 카드는 유지) 새 클럽 선수단이 지급되고 베스트 XI가 다시 배치됩니다.
