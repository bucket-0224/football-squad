# 시작하기

## 로컬 실행

```bash
npm install
npm start
# → http://localhost:3000
```

- 기본 포트는 3000이며 `PORT` 환경변수로 변경할 수 있습니다.
- 데이터는 `backend/data/db.json`에 저장됩니다(서버가 자동 생성).
- 선수 카드 이미지는 선택 사항입니다:
  ```bash
  node scripts/fetch-player-images.js   # 렌더/컷아웃 다운로드
  python3 scripts/crop-upper-body.py    # 상체 크롭 정규화 (PIL 필요)
  ```

## 회원가입

1. 아이디·비밀번호·구단 이름을 정합니다.
2. **시작 팀**을 고릅니다 — 8개 리그의 모든 클럽 또는 국가대표팀. 큐레이션 클럽은 손질된 로스터, 그 외 클럽은 실제 선수 명단을 실시간으로 불러옵니다.
3. 시작 팀 선수단 전원이 지급되고, 베스트 XI가 자동 배치된 상태로 시작합니다.

## 재화

| 재화 | 획득 | 사용처 |
| --- | --- | --- |
| 🪙 코인 | 경기 보상, 예측 적중, 중복 카드 환급, 선수 판매 | 팩 뽑기, 이적 협상, 강화 |
| 🏆 승점 | 랭크 매치 승/무 | 클럽 변경(250), 랭킹 |

## 두 개의 스쿼드

- **클럽 스쿼드** — 보유한 모든 선수 배치 가능. AI 연습 경기에 사용.
- **실전 스쿼드** — **뽑기로 획득한 카드만** 배치 가능. 랭크 매치(유저 대전)에 사용. 빈 슬롯은 유스(OVR 40)가 대신 뜁니다.
