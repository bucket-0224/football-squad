# 경기 규칙 (EPL 기반)

탑뷰 중계는 실제 EPL(IFAB) 규칙을 기반으로 연출됩니다.

## 재개 규칙

| 상황 | 처리 |
| --- | --- |
| 킥오프 | 경기 시작은 홈, 실점 후엔 먹힌 팀, 후반은 어웨이 |
| 하프타임 | 45분에 진영 교대, 어웨이 킥오프 |
| 스로인 | 상대가 마지막으로 건드린 공이 터치라인을 넘으면, 우리 선수가 라인 밖에서 던짐 |
| 골킥 | 공격팀 최종 터치로 골라인 아웃 → 키퍼가 박스 안에서 자유롭게 전개 (첫 배급까지 압박 금지) |
| 코너킥 | 수비 최종 터치로 골라인 아웃 → 나간 쪽 코너 아크에서 재개. 박스 자리싸움, 골사이드 마킹, 숏코너, 헤더 경합(클리어/키퍼/골킥) |
| 오프사이드 | 패스 순간 리시버가 최종 수비수·공보다 앞(상대 진영)이면 휘슬 → 수비팀 프리킥 |
| 추가시간 | 후반 종료 후 2~5분 (`90+n'` 표기) |

## 파울 & 카드

- 파울은 실제 태클 접촉 시 휘슬 — 박스 밖은 프리킥(수비벽 3인), 박스 안은 페널티킥(키커·키퍼 외 박스 밖 대기)
- **어드밴티지**: 파울당한 팀이 유리한 위치면 휘슬 없이 플레이 온
- **옐로카드** 누적 2장 = 퇴장, **심각한 반칙은 즉시 퇴장** — 퇴장 팀은 10명으로 싸우고(기대 득점 감소) 그 선수는 재출전 불가
- **백패스 룰**: 키퍼는 발로 온 백패스를 손으로 잡을 수 없어 즉시 발로 처리
- **VAR**: 골망을 갈랐어도 판독 끝에 오프사이드로 취소될 수 있음

## 페널티/프리킥 확률

실제 EPL 통계에 맞춰 조정되어 있습니다 — 페널티 전환율 ~76%, 경기당 골 ~2.9, 레드카드 ~0.2회.
